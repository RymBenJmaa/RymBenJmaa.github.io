Thakarhom

Une application IOS

❮ ❯



Une application pour faciliter la vie de tous les jours aux personnes qui s'occupent des personnes atteinte de la maladie d'Alzheimer à travers une liste de symptômes des conseils pour les problèmes quotidiens la position de la personne atteinte et le chemin pour y aller...
En travaillant sur ce projet, j'ai utilisé Swift4 comme language de programmation, Laravel pour le dévéloppement des web services et MySQL comme système de gestion de base de données. J'ai intégré divers API, telles que Google Maps pour le repérage du malade et les directions pour y aller, une API pour la carousselle de conseils, ainsi que d'autres afin de fournir une meilleure expérience à l'utilisateur.

  • Date: Janvier 2018
  • Client: ESPRIT
